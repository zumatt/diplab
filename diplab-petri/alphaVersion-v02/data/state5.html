DiPLab

To analyze in a better way how the different antibiotics react to the bacteria inserted, can be useful to zoom-in to see the resistance.

In microscope mode you will have the possibility to see each AB inserted in the DiPLab Petri in 4 different steps: 0x, 4x, 10x and 40x. In these steps you will have the possibility to see how a single antibiotic react to the bacteria previously inserted.
